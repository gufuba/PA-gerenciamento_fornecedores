Aplicação Spring Boot para gerenciar fornecedores. O objetivo é
praticar mapeamento JPA com campos específicos de negócio e operações CRUD.

Endpoints RESTful: 
POST /fornecedores — criar novo fornecedor |
GET /fornecedores — listar todos os fornecedores |
GET /fornecedores/{id} — buscar fornecedor por ID |
DELETE /fornecedores/{id} — deletar fornecedor por ID
